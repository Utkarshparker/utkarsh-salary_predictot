# utkarsh-salary_predictot

1. **NAME**: Utkarsh Maan Mishra
2.  **PROJECT TITLE**: Salary prediction using regression
3.  **PROJECT DESCRIPTION**: This project predicts player salaries using regression based on performance statistics such as points, assists, rebounds, and games played from the National Basketball Association dataset. Data analysis and visualization are used to understand relationships between player performance and salary, and a regression model is applied to estimate salary values.
4.   **Methodology**:The dataset from the National Basketball Association is first loaded and preprocessed by cleaning data and selecting important features such as age, points, assists, and rebounds. Exploratory data analysis and visualizations are used to understand relationships between variables. Then a regression model is trained on the dataset to predict player salaries based on performance statistics, and the model's predictions are analyzed.
5.** RESULTS** : The regression model was able to predict player salaries based on performance statistics such as points, assists, and rebounds from the National Basketball Association dataset. The analysis showed that players with higher performance metrics generally tend to receive higher salaries, indicating a positive relationship between player performance and salary.
